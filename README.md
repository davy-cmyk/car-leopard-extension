# Car Leopard Insight (Extension MVP)

## What it does
- Detects VIN on listing pages
- Shows a small “Car Leopard Insight” badge
- Clicking opens https://carleopard.com/?vin=...

## Local test (Chrome)
1. Go to chrome://extensions
2. Enable Developer mode
3. Load unpacked → select this folder
4. Visit a listing page with a VIN
